# javascript-web-catholic-holidays

## Description
A small project to demonstrate using
api, bootstrap, and datatable. The datatable
is searchable, sortable, and paginated.

## Tech stack
- jquary 3.5
- calapi

## Docker stack
- httpd:latest

## To run
`sudo ./install.sh -u`
Availble at http://localhost

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
